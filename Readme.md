# 🌟 Interactive Responsive Landing Page

A modern and fully responsive landing page featuring an interactive fixed navigation menu with scroll-based animations, hover effects, smooth transitions, and engaging UI interactions.

## 🚀 Features

* ✅ Fixed Navigation Bar
* ✅ Scroll-Based Navbar Style Change
* ✅ Hover Effects on Navigation Links
* ✅ Active Section Highlighting
* ✅ Scroll Progress Indicator
* ✅ Responsive Mobile Navigation Drawer
* ✅ Smooth Scrolling Experience
* ✅ 3D Card Tilt Animations
* ✅ Hero Section Parallax Effects
* ✅ Interactive UI Components
* ✅ Fully Responsive Design
* ✅ Built with Pure HTML, CSS & JavaScript

## 🎯 Project Objective

This project was created as part of my internship task to design an interactive navigation menu that:

* Changes style/color while scrolling
* Includes hover effects on menu items
* Maintains a fixed position across the page
* Provides a responsive user experience on all devices
