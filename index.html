About

Why a great nav matters

First impressions are made in milliseconds. A navigation bar that responds to user behaviour signals quality — boosting trust and exploration.

🎯

Scroll Awareness

The nav instantly detects scroll position and shifts from transparent to a frosted-glass pill with a subtle shadow.

✨

Active Highlighting

Links auto-highlight based on which section is in view, giving users instant orientation on the page.

📱

Mobile Drawer

A silky hamburger menu slides open on small screens so the experience is just as polished on mobile.

Features

Everything you need

01

Colour Morphing

Background transitions from fully transparent to a rich semi-opaque dark shade with a smooth CSS transition on scroll past 80 px.

02

Progress Indicator

A thin gradient bar at the bottom of the nav tracks how far down the page you've scrolled — real-time, 60 fps.

03

Hover Underline Magic

Every nav link sports a sliding gradient underline that animates from 0 to full width on hover using a pure CSS pseudo-element.

04

Intersection Observer

Section tracking uses the native Intersection Observer API — no scroll event listeners needed for active-link logic, keeping it butter smooth.

05

CTA Pulse

The "Get Started" button subtly pulses before scroll and snaps to a solid style once the nav background kicks in.

06

Zero Dependencies

Built with exactly one HTML file, one CSS file, and one JS file — no build tools, no npm, no frameworks required.
